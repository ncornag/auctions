# Auction Server

## install

```
npm install
```

## run

```
node server.js
```

## optional

### Livereload

Install https://chrome.google.com/webstore/detail/livereload/jnihajbhpnppcggbcgedagnkighmdlei

```
npm install -g livereload2
livereload ./public
```

### Supervisor

```
npm install -g supervisor
supervisor --ignore public server.js
```
